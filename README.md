# JavaScript Projects

A curated collection of JavaScript applications and interactive utilities built using **HTML, CSS, and vanilla JavaScript**, demonstrating core language concepts, DOM manipulation, event handling, input validation, logic implementation, and user interface interaction.

This collection includes small but practical tools and games that reinforce foundational JavaScript skills and provide working examples of dynamic web applications.

## Live Demos

Many of these projects are deployed via GitHub Pages and include live demo links for easy viewing and interaction.

## Projects Included

### 🧮 Calculator
A fully functional on-page calculator with dynamic display resizing, backspace, error handling, and arithmetic operations.

### 🔢 Counter 
An interactive counter that increments, decrements, and resets values dynamically using event-driven DOM updates.

### 🎲 Dice Roller
Enter a number of dice to roll; this app generates random dice values and displays them instantly.

### 🪙 Random Number Generator
Generates a random number within a user-defined range with instant feedback.

### 🔁 Number Guessing Game
Guess a randomly chosen number within a range selected via difficulty levels — ranges from Easy up to Insane (1–1000).

### 🔐 Password Generator
Instantly generates random passwords via structured logic and dynamic DOM updates.

### 🌡️ Temperature Converter
Convert values between Celsius and Fahrenheit with real-time calculation and result display.

## How to Use

1. Clone or download the repository.
2. Navigate into any project folder.
3. Open the `index.html` file in a modern browser.
4. Use the UI interactions to run and test the project.

Each project runs independently and does *not* require installation of external dependencies.

## Technologies Used

- **HTML5** — structure and semantics  
- **CSS3** — styling, layout, responsive UI  
- **Vanilla JavaScript** — logic, DOM manipulation, event handling

## Why This Repo

This repository is designed to:

- Demonstrate practical use of JavaScript fundamental concepts  
- Provide real working examples of interactive browser applications  
- Serve as a reference and practice space for learners and future developers
